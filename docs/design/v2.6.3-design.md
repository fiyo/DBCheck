## 功能 1：流式输出 + Markdown 渲染

### 现状
- `_call_llm()` 是同步调用，等完整回复生成完才返回
- 前端显示"正在输入..."动画，用户体验差
- AI 回复是纯文本，没有 Markdown 渲染

### 目标
- AI 回复逐字/逐句显示（流式）
- 支持 Markdown 语法渲染（代码块、表格、加粗等）

### 实现方案

#### 后端（`web_ui.py`）

```python
def api_chat_stream():
    """SSE 流式返回 AI 回复"""
    data = request.get_json() or {}

    context = {k: v for k, v in data.items() if k not in ('message', 'session_id')}
    
    def generate():
        # 1. 发送开始标记
        yield f"data: {json.dumps({'type': 'start'})}\n\n"
        
        # 2. 调用 LLM（流式）
        cfg = _load_ai_config()
        backend = cfg.get('backend', 'ollama')
        
        if backend == 'ollama':
            # Ollama 支持 stream=True
            import requests
            payload = {
                'model': cfg.get('model', 'qwen2.5:3b'),
                'prompt': prompt,  # 构建好的 prompt
                'stream': True,
                'options': {'temperature': 0.7}
            }
            with requests.post(
                f"{cfg.get('host', 'http://localhost:11434')}/api/generate",
                json=payload,
                stream=True,
                timeout=60
            ) as r:
                for line in r.iter_lines():
                    if line:
                        chunk = json.loads(line)
                        if 'response' in chunk:
                            yield f"data: {json.dumps({'type': 'chunk', 'content': chunk['response']})}\n\n"
                        if chunk.get('done', False):
                            break
        else:
            # OpenAI 兼容流式
            pass
        
        # 3. 发送结束标记
        yield f"data: {json.dumps({'type': 'done'})}\n\n"
    
    return Response(generate(), mimetype='text/event-stream')
```

**1.5 轻量级 Markdown 渲染器**

```javascript
function renderMarkdown(text) {
  // 代码块：```language\ncode```
  text = text.replace(/```(\w+)?\n([\s\S]*?)```/g, '<pre><code>$2</code></pre>');
  // 行内代码：`code`
  text = text.replace(/`([^`]+)`/g, '<code>$1</code>');
  // 加粗：**text**
  text = text.replace(/\*\*(.*?)\*\*/g, '<strong>$1</strong>');
  // 斜体：*text*
  text = text.replace(/\*(.*?)\*/g, '<em>$1</em>');
  // 标题：### text
  text = text.replace(/^### (.*?)$/gm, '<h3>$1</h3>');
  text = text.replace(/^## (.*?)$/gm, '<h2>$1</h2>');
  text = text.replace(/^# (.*?)$/gm, '<h1>$1</h1>');
  // 列表：- item
  text = text.replace(/^- (.*?)$/gm, '<li>$1</li>');
  // 段落
  text = text.replace(/\n\n/g, '<br><br>');
  return text;
}
```

```python
@app.route('/api/chat', methods=['POST'])
def api_chat():
    data = request.get_json() or {}
    message = data.get('message', '')
    session_id = data.get('session_id', 'default')
    mode = data.get('mode', 'auto')  # 新增
    
    # 根据 mode 决定处理方式
    if mode == 'qa':
        # 强制问答模式
        result = _answer_chat_qa(session_id, message, context=chat_context)
    elif mode == 'inspect':
        # 强制巡检模式
        result = _execute_inspection(session_id, message)
    else:
        # auto：自动判断（原有逻辑）
        intent = _classify_chat_intent(message)
        if intent == 'inspect':
            result = _execute_inspection(session_id, message)
        else:
            result = _answer_chat_qa(session_id, message, context=chat_context)
    
    return jsonify(result)
```

## 技术栈

| 功能 | 技术 | 备注 |
|---|---|---|
| 流式输出 | SSE (Server-Sent Events) | Flask `Response(stream_with_context)` |
| Markdown 渲染 | 轻量级自定义渲染器 | 避免外网 CDN 依赖 |
| 多轮对话 | 会话历史 + LLM 摘要 | `_CHAT_SESSIONS` 内存 + `pro/chat_history.json` 持久化 |
| 模式切换 | 前端状态管理 | `chatMode` 变量 + badge UI |
| SQL AI 辅助 | 复用 `/api/chat/stream` | 专用 prompt + 代码块提取 |

## 风险 & 缓解

| 风险 | 影响 | 缓解措施 |
|---|---|---|
| Ollama 流式输出不稳定 | 功能不可用 | fallback 到非流式 |
| SSE 连接超时 | 长对话断开 | 心跳机制 + 自动重连 |
| Markdown 渲染 XSS | 安全风险 | 转义 HTML + 白名单标签 |
| LLM 摘要消耗 token | 成本增加 | 只在对话 >10 轮时摘要 |
| SQL 优化结果不准确 | 用户体验差 | 在 AI 回复里注明"建议测试后再执行" |
